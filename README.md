# SlideShow Digital Signage System - support

This repository is dedicated to support, discussions, bug reports, and feature requests for the SlideShow app. If you have a question, encounter an issue, or would like to suggest an improvement, please create a new Issue in this repository (see the Issues tab at the top of the page).

You can also submit a bug report directly from the software:
- App's on-screen menu → `Bug report`
- Web interface → menu `Information` → `Bug report`.

**:warning: Note: This repository does not contain source code. Slideshow is free to use but not open source.**

## About SlideShow

SlideShow is a free digital signage system for Android.

Learn more or download the app:
